## LibrePIM Contributors

LibrePIM extends the foundation of Akeneo CE with:

- **Dependency Updates**: Modern package versions (Symfony 6+, Doctrine 3+, Elasticsearch 8+)
- **Security Patches**: CVE fixes and security hardening
- **Bug Fixes**: Community-reported issues and improvements
- **Long-Term Support**: Maintenance beyond Akeneo CE's release cycle
- **Compatibility Layer**: Support for modern PHP and database versions

### How to Contribute

We welcome contributions! Please:

1. **Respect Original Licenses**: All code must comply with OSL-3.0 (or compatible licenses)
2. **Attribute Work**: Clearly document modifications and their purpose
3. **Credit Authors**: Maintain copyright notices for original work
4. **Test Thoroughly**: Ensure backward compatibility where possible
5. **Document Changes**: Update CHANGELOG and code comments
